# 🌦️ Weather Forecasting App

A responsive weather forecasting application developed using **React**.  
This app fetches real-time weather data from an external Weather API and displays current weather conditions along with forecast details in a clean and user-friendly interface.

## 📌 Project Objective

The main objective of this project is to develop a weather forecasting application using React that allows users to view current weather conditions and forecast information by fetching data from an external API.

## ✨ Features

- 🌍 Search weather by city name
- 🌡️ Display current temperature and weather condition
- 💧 Show humidity details
- 💨 Display wind speed information
- 📅 Show weather forecast details
- 🔄 Fetch API data using `useEffect`
- ⚠️ Error handling for invalid city names or API issues
- 📱 Responsive and clean user interface
- 🎨 Styled using CSS / CSS framework

## 🛠️ Technologies Used

- React.js
- Vite
- JavaScript
- HTML5
- CSS3 / Tailwind CSS
- External Weather API

## ⚙️ How It Works

1. User enters a city name.
2. The application sends a request to an external Weather API.
3. Weather data is fetched using React `useEffect`.
4. Current weather details and forecast information are displayed.
5. If the city name is invalid or the API fails, an error message is shown.
